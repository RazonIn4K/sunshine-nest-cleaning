# Marketing assets — Sunshine Nest Cleaning

Collateral for promoting the business. **Not deployed** with the website (kept out of `public/`).

| File | What it is | Use |
|---|---|---|
| `google-business-profile.md` | Google Business Profile setup pack | Copy/paste into <https://business.google.com> |
| `assets/social-square.png` | 1080×1080 branded card | Instagram / Facebook posts, profile/cover |
| `assets/flyer.png` | 8.5×11 print flyer (150 dpi) with tear-off phone tabs | Print for community boards, mailboxes, local businesses |

See also `../docs/seo-launch.md` for Search Console + structured-data submission.

**Copy rule:** all assets use only verified, safe claims (local & family-owned, bilingual, free quotes, flexible scheduling, the service types). No insured/bonded/reviews/guarantee/years-in-business language until those are true.

Regenerate the images by re-rendering the source card HTML and screenshotting at the element sizes (1080×1080 and 1275×1650).
